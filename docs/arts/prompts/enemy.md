# Enemy Sprites

- **Dimensions**: 32×32
- **Target**: `gol-project/assets/sprites/enemies/` (single) or `sprite_sheets/enemies/` (animated)

## Prompt Template

```
A [threat level] [enemy type] creature for a 2D survival game.
[Distinctive feature]. [Movement pose].
Menacing but readable silhouette. Distinct from player character.
```

## Example (Existing: enemy_32x.png)

```
A shambling undead figure with torn clothing and exposed bones.
Lurching forward with arms slightly raised. Glowing eyes.
Menacing but readable silhouette. Distinct from player character.
```

## Animation Frames

| Animation | Frames | Notes |
|-----------|--------|-------|
| idle | 2 | Swaying or twitching |
| walk | 4 | Shambling gait |

## Variations

- Zombie types (fast, slow, armored)
- Mutant creatures
- Different damage states

## Avoid

- Making enemies look too similar to player character
- Overly complex silhouettes
- Bright colors that clash with muted palette
