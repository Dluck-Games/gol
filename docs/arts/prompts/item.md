# Item / Pickup Sprites

- **Dimensions**: 32×32
- **Target**: `gol-project/assets/sprites/items/`

## Prompt Template

```
A [item type] pickup item for a 2D survival game.
[Distinctive shape]. Ground-level perspective.
Clear silhouette, identifiable at small scale.
```

## Examples (Existing)

- `item_box_01.png` — Supply box item (32×32)
- `zombie_basement_01.png` — Building sprite (64×64, exception)

## Naming

- `<item_name>_01.png` — numbered variants

## Avoid

- Making items look like terrain or enemies
- Too much detail for 32×32
- Unclear what the item represents
