# Resource Icons

- **Dimensions**: 16×16
- **Target**: `gol-project/assets/icons/resources/`

## Prompt Template

```
A [resource/status] icon for a 2D survival game UI.
Extremely simple, 2-3 colors maximum. Instantly recognizable.
Must be readable at 16×16 pixels. No fine detail.
```

## Examples (Existing)

- `wood.png` — Wood resource icon (16×16)
- `component_point.png` — Component point icon (16×16)

## Avoid

- Fine detail (16×16 is extremely small)
- More than 3 colors
- Abstract shapes that aren't instantly recognizable
- Text or letters in the icon
