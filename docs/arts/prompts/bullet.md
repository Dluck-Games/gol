# Bullet / Projectile Sprites

- **Dimensions**: 12×12
- **Target**: `gol-project/assets/sprites/bullets/`

## Prompt Template

```
A [projectile type] for a 2D survival game.
[Shape and trail description]. Small and fast-looking.
Clear direction of travel. Minimal detail, maximum readability.
```

## Example (Existing: bullet_normal_12x.png)

```
A simple round bullet projectile for a 2D survival game.
Small bright dot with subtle motion trail. Fast-looking.
Clear rightward direction of travel. 2-3 colors maximum.
```

## Naming

- `bullet_<type>.png` — base size
- `bullet_<type>_12x.png` — explicit 12px variant

## Avoid

- Too much detail (12×12 is tiny)
- More than 2-3 colors
- Unclear direction of travel
- Making bullets too large relative to characters
